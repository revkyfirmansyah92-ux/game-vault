📘 Tutorial: Facebook Auto-Post Setup

Step-by-step guide untuk ambil Facebook Page Access Token

1

Buka Facebook Developer

Buat Facebook App untuk bisa post otomatis

Buka link ini di browser:

https://developers.facebook.com

Login pakai akun Facebook lo yang punya Page.

🌐

Buka developers.facebook.com

Klik "My Apps" di pojok kanan atas

⬇️
2

Buat App Baru

Create App untuk akses Facebook Page API

  1. Klik tombol "+ Create App"
  2. Pilih tipe: "Business" → klik Next
  3. Isi App Name: misal "Game Vault Auto Post"
  4. Isi Contact Email: email lo
  5. Klik "Create App"
➕

Create App → Business → Isi nama → Create

⬇️
3

Buka Graph API Explorer

Tool untuk generate access token

Buka link ini:

https://developers.facebook.com/tools/explorer
🔧

Graph API Explorer — tempat generate token

⬇️
4

Generate Page Access Token

Pilih app & page, lalu generate token

  1. Pilih App: di dropdown kiri atas, pilih app "Game Vault Auto Post" yang baru dibuat
  2. Klik "Generate Access Token"
    User Token → Page Token
  3. Centang permission ini:
    pages_manage_posts pages_read_engagement pages_show_list publish_video
  4. Klik "Generate Access Token"
  5. Copy token yang muncul!
    EAAGm0X4ZCZC... (panjang banget)
🔑

Generate Access Token → Centang permissions → Copy

⬇️
5

Ambil Page ID

ID halaman Facebook lo

Cara 1 — Dari URL Page lo:

facebook.com/GameVaultRewards
atau
facebook.com/profile.php?id=1234567890

Cara 2 — Dari Page Settings:

  1. Buka Facebook Page lo
  2. Klik "Settings" / "Pengaturan"
  3. Scroll ke bawah, cari "Page ID"
  4. Copy angkanya
📄

Page Settings → Scroll bawah → Page ID

⬇️
✓

Kirim ke Gue!

Kasih 2 hal ini, langsung gue setup auto-post

EAAGm0X4ZCZC...xxxxx
1234567890

Setelah lo kirim 2 itu ke gue:

Gue bikin script auto-post yang jalan otomatis setiap hari ke Facebook Page lo! 🚀

💡 Tips Penting

  • ⚠️ Token expired — Page Access Token bisa expire. Gue bisa bikin long-lived token biar tahan lama.
  • ⚠️ Jangan share token — token = akses ke page lo. Jangan kasih ke sembarang orang.
  • ✅ Page, bukan Profile — auto-post cuma bisa ke Facebook Page, bukan personal profile.
  • ✅ Rate limit — Facebook limit 25 post/hari ke page. Gue set 2-3 post/hari aja.
